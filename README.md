# Litto-Watch

Plateforme web pour la gestion de données littorales et marines avec authentification, cartographie, campagnes terrain, indices de télédétection (NDVI/NDWI) et module de prédiction des mares.

## Fonctionnalités

- Authentification utilisateur
- Tableau de bord
- Gestion des espèces
- Facteurs biotiques et abiotiques
- Cartes interactives
- Télédétection NDVI / NDWI
- Prédiction des mares
- Campagnes terrain

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Leaflet

## Lancement

```bash
npm install
npm run dev
